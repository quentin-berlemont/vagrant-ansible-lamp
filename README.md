vagrant-ansible-lamp
====================

This is a simple LAMP stack on vagrant using ansible as a provisioning tool.

Requirements
------------
* [**VirtualBox**][1]
* [**Vagrant**][2]
* [**Git**][3]

Technical Details
-----------------
* Ubuntu 14.04 64-bit
* Apache 2.x
* PHP 7.x
* MariaDB 10.x

Quick start
-----------
```
git clone https://github.com/quentin-berlemont/vagrant-ansible-lamp.git
cd vagrant-ansible-lamp
vagrant up
```
